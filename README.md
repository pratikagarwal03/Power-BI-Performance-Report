Power BI Performance Report project for Plant & Co.

Dynamic dashboard that can compare multiple year data with different KPI's (Gross profit, Quantity & Sales).

Can compare account profitablity using scatter plot and countries with worst KPI's using treemap.

Waterfall chart see increase/ decrease in metrics and can be drilled down for country's & products.
